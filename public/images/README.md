# Placeholder Images Directory

This directory contains placeholder images for the Edone Solutions website.

## Directory Structure

```
images/
├── services/
│   ├── 360/           # 360 Virtual Tour images by category
│   ├── design/        # Website Design sample images
│   ├── development/   # Website Development sample images
│   └── google-ads/    # Google Ads related images
├── case-studies/      # Before/After screenshots for case studies
├── clients/           # Client logo images
└── samples/           # Sample design/development screenshots
```

## Replacing Placeholder Images

1. **360 Virtual Tours**: Add 360 panorama photos and before/after screenshots to `services/360/[category]/`
2. **Case Studies**: Add before.jpg and after.jpg to `case-studies/` 
3. **Client Logos**: Add client logo PNG files to `clients/`
4. **Sample Screenshots**: Add design/development samples to `samples/`

## Recommended Image Formats

- **Photos**: Use WebP or JPEG for photographs
- **Logos**: Use PNG with transparent background or SVG
- **Screenshots**: Use PNG for crisp UI screenshots

## Image Optimization

All images should be optimized before uploading:
- Use tools like ImageOptim, TinyPNG, or Squoosh
- Recommended max dimensions: 1920x1080 for full-width images
- Keep file sizes under 200KB where possible
